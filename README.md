Cara Buka

1. fix path my-sql 
  - Buka configure build path
  - select mysql-connector.jar
  - pilih edit lalu cari file mysql-connector.jar
2. Import database
  - Buka phpmyadmin (localhost/phpmyadmin)
  - Pilih menu import
  - Pilih choose file
  - Pilih file sudutmeong.sql yang di folder Data
  - Pencet tombol go
3. Login

    - username = admin
